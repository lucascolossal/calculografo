# Calculógrafo

Calculadora de precificação para fotografia freelance.

## O que faz
Calcula o valor sugerido para um trabalho fotográfico com base em horas de captura, edição, distância, urgência e complexidade.

## Como usar
Execute `calculografo.py` no terminal com Python 3 e preencha os dados.
